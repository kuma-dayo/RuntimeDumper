# RuntimeDumper

- Tool to dump method RVA address for certain anime game
- Supported Version
  - OSRELWin3.4.0
  - OSRELWin3.5.0
  - OSRELWin3.6.0
  - OSRELWin3.7.0
  - OSCBWin3.7.50
  - OSRELWin3.8.0

## How to use?

- Download dll from release or compile it (**as Release**) yourself
- Rename it to `version.dll` and put it in the same folder as the game
